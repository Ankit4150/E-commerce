# E-Commerce MERN Project

## Project Description
This is a full-stack **E-Commerce application** built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
Users can browse products, add items to the cart, make payments, and admins can manage products and orders.  

The application features user authentication, responsive design, and real-time notifications.

---

## Features
- **User Authentication:** Sign up and login with JWT authentication
- **Product Management:** Browse, search, and filter products
- **Shopping Cart:** Add/remove products and update quantity
- **Admin Panel:** Add, update, or delete products (Admin only)
- **Notifications:** Alerts for success, errors, and other actions
- **Responsive Design:** Works on desktop and mobile

---

## Technologies Used

### Frontend
- **React.js** (v19)  
- **React Router Dom** for routing  
- **Redux & Redux Toolkit** for state management  
- **Tailwind CSS** for styling  
- **React Toastify** for notifications  
- **React Icons** for icons  


### Backend
- **Node.js** & **Express.js** for server  
- **MongoDB** & **Mongoose** for database  
- **JWT** for authentication  
- **Bcrypt.js** for password hashing  
- **Stripe** for payment integration  
- **Nodemailer** for sending emails  
- **Cors** for cross-origin requests  
- **Cookie-parser** for handling cookies  
- **Dotenv** for environment variables
